# RockPaperScissors


An AI to play the Rock Paper Scissors game

## Requirements
- Python 3
- Tkinter


## Set up instructions
1. Clone the repo.
```sh
$ https://github.com/Kakoytobarista/RockPaperScissors.git
$ cd RockPaperScissors
```


2. Play the game with your computer!
```sh
$ python3 rock-paper-scissors.py run
